Usando links externos

Na primeira forma o usuário vai sair do seu site para entrar no repositório, assim aumentando as chances de ele nunca mais voltar, no segundo modelo, quando ele clicar no hyperlink, o repositório vai abrir em outra aba, e seu site vai continuar lá aberto. Quando você for fazer um hyperlink com uma página que não é sua, você deve usar links com target blank e rel external.
